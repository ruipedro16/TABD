\begin{abstract}
This report describes the practical assignment of the Advanced Topics in Databases course. 

This practical assignment consists in creating a data warehouse and conducting data analysis on it, as well as creating graphical reports using the Python library \texttt{matplotlib}.

In this report, we briefly describe our approach to the problem and discuss the decisions
we made.
\end{abstract}
